# Chatbot — DecodeLabs Internship

A simple FAQ Chatbot built using Python and Flask that answers frequently asked questions through a clean web interface.

## 🛠️ Tech Stack
- Python
- Flask
- HTML/CSS (Templates)

## ✨ Features
- Responds to common FAQ queries
- Clean and simple web UI
- Rule-based response system

## 📁 Project Structure
chatbot/
├── app.py
├── templates/
│   └── index.html
└── README.md

## 🚀 How to Run
1. Clone the repo
git clone https://github.com/Rashmitha0408/chatbotTask1-RashmithaSai.git
2. Install dependencies
pip install flask
3. Run the app
python app.py
4. Open http://127.0.0.1:5000
